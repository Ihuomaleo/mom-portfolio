# Personal Portfolio Website for Dr. Jane Obasi

This is a personal portfolio website for Dr. Jane Chinelo Obasi, a Senior Lecturer, Editor, and Researcher. The website is a modern, single-page application that is fully responsive and accessible.

## Features

* **Single-Page Scrolling:** Smooth scrollspy navigation that highlights the active section in the sidebar.
* **Responsive Design:** Adapts fluidly from large desktop monitors down to mobile devices.
* **Theme Switcher:**
    * **5-Color Options:** Allows users to change the primary "skin" color of the site instantly.
    * **Dark Mode:** A persistent light/dark mode toggle that remembers the user's choice in their browser.
* **Dynamic Content:** Features a "typing" animation for the hero section and a dynamic "Publications" section populated from Dr. Obasi's CV.
* **Accessibility (A11y):** Built with accessibility in mind, using semantic HTML, ARIA attributes, and keyboard-navigable controls (like the buttons in the style switcher).
* **Functional Contact Form:** Includes an HTML-based contact form ready to be connected to a backend service like Formspree.
